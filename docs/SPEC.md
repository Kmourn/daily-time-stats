# 日记功能规格

## 程序形态

- Ubuntu 桌面程序，窗口界面运行。
- 中文显示名：日记。
- 英文包名：daily-time-stats。
- 发布格式：`.deb` 安装包。
- 技术方案：Python、PySide6、PySide6.QtCharts、SQLite。
- 适配目标：Ubuntu 20.04。

## 页面

- 顶部标签页：
  - 今日时间统计
  - 一周时间统计
  - 一月时间统计
- 备份入口：
  - 导出备份
  - 恢复备份

## 今日时间统计

- 程序打开默认显示今天数据。
- 允许切换日期并调整非本日数据。
- 支持手动录入和计时器。
- 计时器先选择事项和内容，再开始计时。
- 计时器只包含开始、结束两个动作。
- 时间点精确到秒，按 24 小时制显示，例如 `14:29:53`。
- 条目字段：
  - 事项：固定为调研、学习、阅读、代码、写作、沟通、休息。
  - 内容：用户自拟，按事项单独记忆旧名称，预留增加/删除记忆接口。
  - 开始时间-结束时间：允许同一条目新增多段。
  - 备注。
- 一个条目下可包含多段时间，总时长合并计算。
- 时间段数据预留“同一条目第几段时间”的字段。
- 支持编辑条目、删除条目、删除某一段时间、给已有条目新增时间段。
- 允许跨日，统计时按实际日期拆分。
- 允许时间段重叠，统计时重复累计。
- 底部有“今日事项完毕”按钮。
- 底部小字显示“本日数据已备份/本日数据未备份”。

## 备份

- 本地 SQLite 数据库。
- 支持手动导出备份。
- 支持从备份恢复。
- 点击“今日事项完毕”后自动备份数据。
- 同日自动备份只保留最后一次数据，当日旧备份覆盖。
- 自动备份只保留最近 7 份。
- 数据结构注意 Ubuntu 与 Windows 之间可迁移。

## 一周时间统计

- 周一作为一周开始，周一到周日为一周。
- 默认显示所选日期所在周。
- 允许切换不同周。
- 无记录数据按 0 显示。
- 本周总时长：一句话居中，按“小时-分钟”显示。
- 事项用时：柱状图，横坐标按调研、学习、阅读、代码、写作、沟通、休息排列。
- 每日用时：折线图，横坐标按周一至周日排列。
- 底部按钮：“与上周对比”。
- 单击对比后：
  - 总时长区域上侧显示上周总时长。
  - 事项用时每个横坐标显示两个柱，左侧上周，右侧本周。
  - 每日用时同时显示上周和本周折线图。

## 一月时间统计

- 默认模式为“按日统计”。
- 允许切换不同月份。
- 本月总时长：一句话居中，按“小时-分钟”显示。
- 事项用时：柱状图，横坐标按调研、学习、阅读、代码、写作、沟通、休息排列。
- 每日用时：默认按星期聚合，横坐标为周一至周日。
- “按周统计”模式：
  - 只统计周一、周日都在本月内的完整周。
  - 本月总时长保持不变。
  - 事项用时：每个事项横坐标同时显示 K 个柱，K 为纳入统计周的数量，从左到右为第一周至第 K 周。
  - 每日用时：同时显示 K 个完整周各自的折线图。

## 时长规则

- 时间点精确到秒。
- 统计显示按分钟向上取整。
- 今日页条目时长也按分钟向上取整。
- 总时长显示为 `X小时Y分钟`。
- 图表数据按小时计算。
- 图表标签显示为 `1h`、`1h30min`、`2h`。
- 图表坐标轴按 `0.5h` 或合适的半小时倍数显示，不显示非 0.5h 的刻度。

## 托盘和关闭行为

- 不做开机自启动。
- 开发托盘常驻、最小化到任务栏。
- 托盘菜单：
  - 显示日记
  - 当前计时状态
  - 结束当前计时
  - 退出
- 点击最小化按钮时最小化到任务栏。
- 没有计时器运行时，点击关闭按钮直接退出程序。
- 计时器运行时，点击关闭按钮提示“计时程序正在运行，是否直接退出程序？”。
- 若确认退出，结束当前计时，保存为当前条目的一段时间，结束时间为退出时刻，然后退出程序。
